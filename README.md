# Presente DBD

Site estatico tematico inspirado em Dead by Daylight para revelar um codigo de presente da Steam.

Deploy publicado na Netlify:
https://presente-dbd-paloma.netlify.app

## Como trocar o codigo

No arquivo `index.html`, altere o valor de `data-code`:

```html
<main class="locker-scene" data-code="COLOQUE-O-CODIGO-AQUI">
```

Depois publique novamente para atualizar o link.
